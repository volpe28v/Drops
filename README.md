# Drops 
川の防災情報を検索

![image](https://user-images.githubusercontent.com/754962/99628718-0f89d500-2a7a-11eb-8dbc-70ba0f2c5858.png)

### Technology
* Vue.js
* cheerio-httpcli
* axios
* mongodb

### Ready
- install mongodb

```
$ npm install
```

### Build
```
$ webpack
```

### Run
```
$ node app.js -p 3000 -i 300 -r 100 -h http://localhost
```
